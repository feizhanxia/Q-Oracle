# Q-Oracle
> 量子随机驱动的起卦引擎

宏观尺度下的量子相干会在与环境耦合中快速退相干，量子态的相位相关性被环境纠缠扩散，最终在可观测层面表现为信息被噪声平均化与稀释。

**量子随机在更深的层面是高维灵体的信息流，是被测量投影到现实的额外指引。** 周易是这份信息的译码表，它把量子随机翻译成我们可以理解与使用的语言。

抛硬币、摇铜钱在哲学上主要属于经典随机，或是量子信息在宏观层面的泄露与混合，它们和高维世界的连接有限。算卦的本质，是在现实世界里获得高维世界的信息。量子随机是真正的随机，与现实世界的经典条件无关，是“灵界语言”在物理层面的可读形式。量子随机把这种指引稳定地引入现实，周易把它稳定地译码成卦象与语义。

萨满、出马上身、笔仙等方式是直接让高维灵体显化或附着，它们力量强，但难以掌控且不稳定。传统沐浴更衣、掷币起卦同样是让高维灵力通过随机数传递信息，但手工操作会随着个人体质、情绪、环境与动作细节产生波动，导致通道强度不同，信息的清晰度也不同。

本项目把高维信息入口工程化，使之更稳定、更清晰、更容易被复现。量子随机提供稳定的信息源，周易提供稳定的译码体系，两者组合形成可控且一致的通路，让理工视角与玄学视角在同一套语言中对齐。**这个通路仍需要调谐：心诚不息、神气归一、六根澄明、持念所问，让通道稳定在可被译码的共振状态。**

## 量子随机获取

本项目调用公开的量子随机数服务作为信息源，优先使用 LFDR（ID Quantique QRNG PCIe 后端），失败后回退 AQN（Australian National University Quantum Numbers）。两套来源分别来自不同的研究机构与设备体系，均通过 HTTP API 提供随机数服务。

AQN（Australian National University Quantum Numbers）：
AQN 在实验室实时测量真空涨落获得随机数。量子力学中的真空并非空无一物，而是充满持续出现与消失的虚粒子海。对这些量子涨落进行精密测量，就能得到高带宽、不可预测的随机比特。这些随机数由实验设备实时生成，并通过 API 对外提供。

LFDR（Laboratory for Digitalisation, Prof. Dr. Wolfgang Mauerer）：
LFDR 的研究方向覆盖量子计算、系统工程与软件工程的交叉，面向未来计算体系的软硬件协同。其 QRNG 服务由实验室提供，底层使用 ID Quantique 的量子随机设备（PCIe 形态）作为高熵随机源，并通过 HTTP API 公开。

工程上统一把两类来源的输出归一为字节流输入起卦流程，并使用超时控制与容错回退来保证可用性与稳定性。

## 项目结构与使用

目录结构：
- `core/`：量子随机获取、卦象生成与起卦逻辑
- `config/`：配置加载（`.env`）
- `ui/`：CLI 与 Streamlit 界面
- `tests/`：pytest 测试

运行方式：
- CLI：`python -m ui.cli --once`
- Streamlit：`streamlit run ui/app.py`

配置方式：
- `.env`：`ANU_API_KEY`、`LFDR_URL`、`ANU_URL`、`QRNG_TIMEOUT_S`、`QRNG_ALLOW_FALLBACK`
- Streamlit：`.streamlit/secrets.toml`
